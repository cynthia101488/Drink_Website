## Drink_Website 飲料訂購網站
> ###### NCU_MIS_SA 「系統分析與設計」之期末專案

<br> 1、請先透過XAMPP啟動Apache, MySQL, Tomcat。
<br> <img src="https://i.imgur.com/QbxDBIK.jpg" width="35%" height="35%">
<br> 2、下載整個專案檔案後，需先將 ***"sa.sql"*** 匯入phpMyAdmin資料庫中，網頁才能連動並正常顯示。
<br> 2、請從 ***"balance.html"*** 開啟，網頁將會顯示首頁，即可進行操作。
<br> 3、如需登入會員以及管理者的帳號密碼，請進入資料庫中查詢，共有五組會員資料及一組管理者資料。
